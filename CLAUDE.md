# CLAUDE.md — проект Armenia Guidebook

Контентный сайт-справочник по Армении — **сейчас технически EN-only** (en, на корне `/`), **план — полный билингв EN + ru** (весь сайт, решение владельца 2026-07-01; раздормантить ru при наполнении — см. блок ниже): достопримечательности, города, еда (директория проверенных мест), маршруты, транспорт, аренда авто, страховки, развлечения, релокация, новости. Главный туристический кластер — **монастыри** (Хор Вирап, Татев+канатка «Крылья Татева», Гарни/Гегард, Нораванк) + природа (озеро Севан, Дилижан, Арагац) + столица Ереван. Места отбираются по высокому рейтингу (>4★ из нескольких источников) и сверяются с опытными путешественниками, гидами и местными жителями. Диверсифицированная партнёрская монетизация (туры, авто, отели, eSIM, страховки, трансферы). **Приоритет — EN/Tier-1** (западные туристы и номады). Единственный источник правды по продукту, нише и приоритетам — **SPEC.md**. Этот файл — правила работы внутри репозитория. При конфликте приоритет: SPEC.md → CLAUDE.md → всё остальное.

## Стек и принципы
- **Astro 6 + Content Collections**, полная статика (SSG). Никаких SSR, БД, CMS.
- **Tailwind CSS 4**. Дизайн-токены — только из `src/styles/tokens.css` (SPEC §10). Хардкод hex-цветов в компонентах запрещён.
- **Без React.** Интерактив — нативный JS внутри Astro-компонентов (карта Leaflet, мобильное меню, лайтбокс).
- Хостинг: **Cloudflare Workers (Static Assets)**. Деплой = push в `main` (Workers Build: `npm run build` → `wrangler deploy`). Превью-деплои на ветках.
- Изображения: webp/avif через `astro:assets`, lazy-load, каждая ≤ 200 КБ.

## Железные правила
1. **Язык — только английский (на старте).** Сайт **одноязычный: en на корне `/`**. Весь новый контент создаётся ТОЛЬКО на en (`src/content/<коллекция>/en/<slug>.md`). `LANGS = ['en']` в `src/i18n/types.ts` и `src/content.config.ts`. hreflang страницы — self `en` + `x-default`. Переключателя языка нет. **ru-движок дормантный** (`ru.ts.dormant`, исключён из tsconfig) — оживить позже флипом `LANGS=['en','ru']` (см. SPEC «ru fast-follow»).
2. **Партнёрские ссылки** — только через `/go/{partner}?c={slug}` с `rel="sponsored nofollow noopener"`. Прямые URL партнёрок в контенте запрещены. На статью 1–3 AffiliateBox. Приоритет связок (план §4): **localrent** (авто) · **trip-tours** (туры — по Армении рабочий) · **yesim** (eSIM) · **kiwitaxi+aviasales** (Кавказ-комбо/аэропорт) · **trip-hotels** (отели) · **safetywing** (страховка). EKTA из набора **удалена** (сломанный URL). **bikesbooking** — НЕ вешать на горные хайки.
3. **URL не меняются** после индексации. Ошибся в slug — 301 в `public/_redirects`, а не переименование.
4. **Факты не выдумывать.** Цены, адреса, телефоны, условия — только из брифа `briefs/{slug}.md` или надёжного первоисточника. Нет данных → `<!-- TODO: факт от владельца -->` и сказать об этом. Рейтинги/отзывы внешних источников дословно не перепечатывать (ToS §15).
5. **YMYL-осторожность (визы/налоги/ВНЖ/границы).** Факты ТОЛЬКО из официальных первоисточников (mfa.am, migration.am, src.am, e-gov.am) + видимая дата проверки + дисклеймер. Известные ловушки Армении (из плана, обязательно при написании): **виза** — развести постоянный безвиз (~68 стран до 180 дней) и временное освобождение для держателей ВНЖ (истекает 01.07.2026, НЕ подавать как общий крючок); **граница с Турцией/Азербайджаном** закрыта с 1993, но идёт сближение 2026 — только датированный статус; ценовые/тарифные якоря (тур Khor Virap–Noravank–Tatev, Ереван→Тбилиси) — диапазоном, фактчек на момент публикации.
6. **Перф-бюджет (мобайл):** LCP < 2.0 с, CLS < 0.1, JS на странице < 50 КБ. Любая новая зависимость тяжелее 10 КБ — спросить владельца.
7. **Доступность:** контраст ≥ 4.5:1, видимый focus, alt у каждого изображения, рабочая навигация с клавиатуры.
8. **SEO-обвязка каждой страницы:** title ≤ 60 зн., description ≤ 155, canonical, hreflang self `en` + `x-default`, OG-картинка, schema.org по типу шаблона (SPEC §14). Меты считать в символах, не байтах.
9. **Сторонние скрипты:** только одна аналитика (GA4, SPEC §17). Никаких виджетов соцсетей, онлайн-чатов, тег-менеджеров. **GA4 ID — ПЛЕЙСХОЛДЕР `G-PENDING-ARMENIA`** (`public/js/ga-init.js` + BaseLayout): владелец заводит реальное свойство ДО запуска. Consent Mode v2 (default denied + баннер CookieConsent), CSP без инлайн-sha256.
10. **Доки не отстают от кода (правило владельца).** После КАЖДОЙ доработки и ДО коммита синхронизировать документацию (PROGRESS/HANDOFF/AUDIT/KALENDAR/ROADMAP — когда заведены; при изменении правил — SPEC/CLAUDE) и закоммитить+запушить в `main` — без напоминаний.

## Контент-модель (гео под Армению)
- **Регионы (`REGIONS`, фильтр каталога):** 3 макрорегиона (10 марзов + Ереван сгруппированы) — `central` (Ереван+Арарат/Армавир/Котайк/Арагацотн), `southern` (Вайоц Дзор/Сюник), `northern` (Гегаркуник/Тавуш/Лори/Ширак). Менять — синхронно в `content.config.ts` + `i18n/types.ts` + словарях (gate `check-enums`).
- **Города-базы (accessFrom / городские статьи):** Ереван (столица), Гюмри, Дилижан, Горис, Джермук; Татев/Арени/Севан — достопримечательности-якоря, не города. Точки въезда (accessFrom): Ереван (Звартноц), Гюмри, Дилижан. Города директории еды: Ереван/Гюмри/Дилижан.
- **Город = статья `category=cities`** (английский слаг). Отдельная коллекция `cities` НЕ используется для наполнения (папки пусты).
- **Валюта:** армянский драм (**AMD**); для Tier-1-читателя можно показывать примерный USD/EUR (точка решения владельца — см. SPEC).
- **Кухня (`CUISINE_KEYS`):** armenian/caucasian/grill(хоровац)/wine/brandy/cafe/bakery/vegetarian/street/bar (без `seafood` — страна без выхода к морю).

## Стандарты контента (объём, фото, тон)
- **Объём.** Статья — **1200–2000 слов** (первый абзац — прямой ответ; H2/H3 под вопросы; без «воды», без FAQ). Новость — **1500–2000 знаков** (~250–350 слов, 3–5 разделов, **≥4 внутр. ссылки**).
- **Фото (минимум).** Статья ≥5 (cover + ≥4 инлайн); новость ≥2; маршрут — cover + фото **каждой остановки**; город ≥10; **столица Ереван ≥15**. Каждое фото уникально, **webp ≤200 КБ**, своё или CC/PD (Wikimedia/Openverse/Unsplash) с атрибуцией.
- **Тон.** Живой, человечный нативный английский. Не ИИ-генерёнка (жёсткий гейт). Факты только из источника; даты/числа — только из источника текущего года.
- **Перелинковка.** Обязательна: статья ≥2 внутр. ссылки / новость ≥4, в обе стороны, без сирот. Кросс-сайт — контекстно (для Армении ключевой сосед — **Грузия**, Кавказ-комбо Ереван↔Тбилиси).
- **Уникальность намерения (④ гейт).** Одна интент-страница: без mirror/variant-дублей; гейт `check-dedup` в `npm test`/`qa`. Бейслайны (`.dedup-baseline.json`/`.photo-baseline.json`) — **пустые** (нет легаси-долга).

## Контент-конвейер (SPEC §12)
`briefs/{slug}.md` (ключевой запрос EN-first, вопросы читателя, ФАКТЫ/первоисточник, цены, фото) → **скелет:** `npm run new -- <тип> <slug> --title "…"` (генерит en + папку фото + DoD) → черновик en по шаблону → фото (CC/свои) → **fact-check → content-editor** → коммит, draft снимается. Для YMYL (визы/налоги/границы) fact-check по первоисточникам обязателен.

## Off-domain статус (важно)
Сайт строится **off-domain**: `public/robots.txt` пока `Disallow: /` (индексация закрыта до готовности). К запуску — вернуть `Allow: /` + `Disallow: /go/`. Перед запуском заменить плейсхолдеры: **GA4-ID** (`G-PENDING-ARMENIA`→реальный), **флаг/иконки** (`am-flag.svg`, `public/icons/*`, `favicon.svg` — сейчас кадры/флаг донора), **hero-фото** (`src/assets/hero/placeholder-*.jpg` → армянские CC), **og-default.jpg**.

## Чего НЕ делать
- Не добавлять React, CMS, базы данных и платные сервисы без явного решения владельца.
- Не клонировать инструменты внутрь репозитория — они подключаются как плагины Claude Code.
- Не генерировать массово страницы без уникальной фактуры (SEO-генерёнка = риск бана, §15).
- Не трогать логику `/go/` без обновления таблицы партнёрок в SPEC §16.
- Не публиковать YMYL-факты (визы/налоги/ВНЖ/границы) от себя — только из официальных первоисточников + дисклеймер (правило 5).
